# NovaPlay Hub

A polished, downloadable game hub inspired by the feel of a modern social gaming platform.

## Run locally

- Double-click launch-gamehub.cmd to open the experience in your browser.
- Or open index.html directly.

## Files

- index.html — main application shell
- styles.css — premium visual design
- script.js — 300-game catalog and filtering
- launch-gamehub.cmd — Windows launcher
also make sure to get pythoin not old
